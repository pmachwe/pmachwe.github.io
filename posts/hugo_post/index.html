The current setup is created following the instructions in this video.

A quick summmary of the steps required to create a new post for this blog.

  1. Create a new post using hugo new posts/<post_name>.md and update it.
  2. Run hugo server to check the contents locally.
  3. Run hugo -t hugo-paper to generate the static page in public directory.
  4. Go to public directory and push the changes.
